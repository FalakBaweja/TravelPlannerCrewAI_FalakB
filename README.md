# Multi-Agent Travel Planner System

This project is an autonomous Multi-AI Agent Travel Planning system built using **CrewAI** and **Groq (LLaMA 3.3)**. 

## Agents Involved
1. **Travel Preference Analyst** - Analyzes user inputs to build a traveler profile.
2. **Destination Research Specialist** - Discovers landmarks and hidden gems.
3. **Transportation Planning Specialist** - Optimizes travel routes and travel times.
4. **Accommodation and Budget Specialist** - Allocates budgets and maps out places to stay.
5. **Itinerary Generation Specialist** - Builds a seamless day-by-day itinerary.
6. **Safety and Risk Assessment Specialist** - Identifies weather risks and travel advisories.
7. **Travel Report Generation Specialist** - Compiles everything into a clean final dossier.
